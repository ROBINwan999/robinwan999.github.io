Hugo搭建个人博客教程

安装hugo

Hugo是一个Go语言编写的开源框架，用于快速生成个人博客，这篇教程用于教大家如何快速搭建自己的博客

首先，需要安装Hugo，如果是Mac的话可以用brew一键安装，如果是Linux或者Windos也可以去官网上下载对应的版本

# Mac
brew install hugo
# Linux / Windows
https://gohugo.io/installation/
# 下载完成后可以使用 hugo version 来验证是否安装成功

新建项目

接下来使用Hugo来创建一个叫myblog的项目

hugo new site myblog

这时，当前目录下会出现一个叫myblog的文件夹，下一步我们来配置一个主题

主题

关于主题，可以在下方的网址中选择一个喜欢的主题，然后通过git clone将其下载到themes中，比如这里将名为m10c的主题clone到了themes中

https://themes.gohugo.io/
# 当前所在文件夹myblog
git clone https://github.com/vaga/hugo-theme-m10c.git themes/m10c

启动博客

本地启动博客很简单，只需要在myblog目录下，输入如下命令即可启动

# 这里 -t 是主题的意思，后面跟的是主题的名字，例如我选择主题叫m10c，最后是启动的意思
hugo server -t m10c --buildDrafts
# 成功启动的话，会看到如下提示
...
Web Server is available at http://localhost:1313/ (bind address 127.0.0.1)
...

到这里就可以访问本地博客了，我们打开浏览器，输入提示信息最后一行给的地址和端口

以上方提示信息为例，我在浏览器输入

127.0.0.1:1313

然后就能看到博客页面

第一篇博客

这个时候发现，虽然博客搭建成功了，但是没有任何文章。让我们来创建第一篇文章

进入myblog目录下，然后通过如下命令来在post目录下创建一个名为blog的文章

hugo new post/blog.md

然后，hugo就会在content/post目录下创建名为blog.md的文件，接下来就可以写我们的第一篇博客了，这里我直接将blog.md的内容贴在下方

---
title: "我的第一篇博客"                         
author: "我是作者"  
description : "这是描述信息"    
date: 2022-11-22        
---


# first blog

## today is good day

### i create my first blog

---

now let's write a basic hello world program

```C
#include <stdio.h>

int main() {
	printf("Hello World!\n");
	return 0;
}
```

关于md，这是一种轻量级标记语言，经常用于程序员记笔记，写文章，可以通过vim或者vscode灯编辑器来进行编写

这里需要注意的是，hugo的md文件和其他md不一样的是，需要在md顶部加入一些信息才能被识别到，比如这里加入的是title, author, date等等

部署到公网

下一步就是利用github或者gitee之类的托管平台来部署我们的博客到公网，从而可以让其他人进行访问，这里以github为例

首先，我们需要在github新建一个仓库，这里的仓库名一定要是自己github账户名一致名切小写，后面加上.github.io，如下例

创建好仓库之后，我们将项目部署到该仓库下，输入如下命令即可

# myblog目录下
hugo --theme=m10c --baseUrl="https://robinwan999.github.io/" --buildDrafts

之后就能看到hugo在当前目录下生成了一个名为public的文件，接下来需要将该文件夹传到远程仓库

# 进入public目录下
cd public
# git 操作
git init
git add .
git commit -m "我的hugo博客第一次提交"
git remote add origin https://github.com/ROBINwan999/robinwan999.github.io.git
git push -u origin master
# 输入github用户名和密码如果被问的话

成功推到github远程仓库之后，我们打开浏览器，刷新一下，就能看到项目的内容了

然后我们复制网址的这一段

robinwan999.github.io.git

就可以通过公网访问我们的个人博客啦
